❓ PRIMERAS PREGUNTAS
═══════════════════════════════════════════════════════════════════════════════

P: ¿Qué hago si recibo error "Cannot find module 'express'"?
R: Ejecuta: npm install (nuevamente en la raíz)

P: ¿Qué si PostgreSQL no conecta?
R: Verifica: psql -U postgres
   Si no abre, instala o inicia PostgreSQL

P: ¿Qué si el puerto 3000 está ocupado?
R: Cambia PORT en .env a 3001 (o el que prefieras)

P: ¿Necesito ejecutar migrations de BD?
R: Depende del schema. Ver src/controllers/ para crear tablas si es necesario

P: ¿Cómo cambio los datos de la compañía?
R: Edita config.json en la raíz del proyecto
